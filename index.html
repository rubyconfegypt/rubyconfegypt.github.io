What is Ruby?
Ruby is a programming language of careful balance of beautiful and practical. It has blended parts of (Perl, Smalltalk, Eiffel, Ada, and Lisp) with the aim for “trying to make Ruby natural, not simple” in a way that mirrors life.
"Ruby is simple in appearance, but is very complex inside, just like our human body." - Yukihiro "Matz" Matsumoto
What is RubyConf Egypt?
RubyConf Egypt is intended to be a nonprofit annual conference for promoting Ruby adoption in Egypt. RubyConf Egypt 2017 is the first Ruby Conference to be held in Egypt.
Who Should Attend RubyConf Egypt?

Java Software Engineers
Python Software Engineers

Grad\Undergrad Students
Anyone with an interest in Programming as a Liberal Art
